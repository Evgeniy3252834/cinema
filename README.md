# 🎬 CineMatch — полиглотная рекомендательная система

## 📋 О проекте
Учебный микросервисный проект, демонстрирующий использование разных типов баз данных в одном приложении. Пользователи могут оценивать фильмы, а система рекомендует похожие на основе:
- Векторного сходства (Qdrant)
- Графовых связей (Neo4j) 
- Истории просмотров (PostgreSQL)
- Поведенческих логов (MongoDB)

## 🗄️ Архитектура баз данных

| База данных | Назначение | Модель данных |
|------------|------------|---------------|
| **PostgreSQL** | Транзакционные данные (пользователи, оценки, фильмы) | Реляционная, ACID |
| **MongoDB** | Сырые события/логи поведения | Документная, гибкая схема |
| **Redis** | Кэш, сессии, real-time счетчики | Key-Value, in-memory |
| **Qdrant** | Векторный поиск похожих фильмов | Векторная, эмбеддинги |
| **Neo4j** | Граф связей (актеры, режиссеры, жанры) | Графовая |

## 🔧 Быстрый старт

### Требования
- Python 3.9+
- Доступ к серверам БД (данные в `.env`)

### Установка
```bash
# Клонируем репозиторий
git clone https://github.com/yourusername/cinematch-recommendations.git
cd cinematch-recommendations

# Создаем виртуальное окружение
python -m venv venv
source venv/bin/activate  # или `venv\Scripts\activate` на Windows

# Устанавливаем зависимости
pip install -r requirements.txt

# Копируем пример .env и заполняем своими данными
cp .env.example .env
